# AI Crop Recommendation System

This project recommends the most suitable crop for farmers based on soil nutrients and environmental conditions using Machine Learning.

## Problem
Farmers often do not know which crop is best for their soil. Choosing the wrong crop can reduce productivity and income.

## Solution
This system predicts the best crop using soil parameters such as Nitrogen (N), Phosphorus (P), Potassium (K), temperature, humidity, pH, and rainfall.

## Features
- Predicts suitable crops based on soil data
- Machine learning based prediction
- Simple web interface for input
- Helps farmers make better farming decisions
